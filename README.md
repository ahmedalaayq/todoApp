## Installation
To run this app locally, follow these steps:

1. **Clone the repository:**
   ```sh
   git clone https://github.com/ahmedalaayq/todoApp.git
   cd todoApp
   ```
2. **Install dependencies:**
   ```sh
   flutter pub get
   ```
3. **Run the app:**
   ```sh
   flutter run
   ```
